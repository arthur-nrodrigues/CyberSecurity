# Conceito Geral:

``https://attack.mitre.org/``

O MITRE ATT&CK é, em resumo, a "bíblia" moderna da Cibersegurança. Basicamente é uma base de conhecimento global que mapeia as formas que hackers usam para atacar sistemas, e que de certa forma mudou a forma como seguranças defendem.

E o que isso implica: Atualmente é usada como uma base universal para identificar e defender sistemas. É mais fácil impedir a técnica usada pelo hacker do que impedir o ataque, dessa forma - quebramos o mal pela raiz!
