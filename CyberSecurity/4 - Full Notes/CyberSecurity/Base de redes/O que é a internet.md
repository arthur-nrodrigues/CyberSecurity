# Conceito Geral:

Para entendermos de início essa questão devemos entender que qualquer sistema que "converse" entre si em um curto espaço de tempo deve estar ligado fisicamente, seja por cabos, fibra óptica entre outros.

**Exemplificação da internet:**

![[Pasted image 20260213164126.png]]

**LAN E WAN**:

LAN (Rede Local) e WAN (Rede de Longa Distância) ==diferem principalmente pela abrangência geográfica e velocidade==. LANs conectam dispositivos próximos (casa/escritório) com alta velocidade e gerenciamento privado. WANs conectam áreas geográficas vastas, como cidades ou países, usando infraestrutura pública (Internet) para unir várias LANs, sendo geralmente mais lentas, ou seja, as WANs são basicamente a internet que conhecemos hoje.

**Endereçamento IP**:
IP é basicamente o CPF da máquina, todo dispositivo que deseja se conectar a outro dispositivo deve obrigatoriamente possuir um endereço ip.

Formato -> 4 números que variam de 0 a 255 divididos por ponto. Exemplo: 167.123.0.10.

**167.123.0. / 10**
Identifica / Identifica
a rede     / o host (dispositivo)

Toda máquina possui um IP próprio, um IP privado e outro IP público.

![[Pasted image 20260213191658.png]]

Geralmente o IP público está armazenado no roteador.
Dessa forma, para que dispositivos em uma mesma rede converse entre si é necessário que estejam obrigatoriamente na mesma rede, identificada pelos três primeiros números do IP.
